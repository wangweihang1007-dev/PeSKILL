# 项目表字段口径

## 列顺序

| 列 | 字段 | 写法 |
|---|---|---|
| A | 序号 | 接续现有序号 |
| B | 项目简称 | 常用简称，不写冗长公司全称 |
| C | 成立时间 | 以工商登记为准，写真实日期 |
| D | 城市 | 注册地城市，通常不写省份 |
| E | 上市申报预期 | 板块、申报/上市年份；仅有计划时加“计划/预计” |
| F | 前一轮融资时点和估值 | 时间 + 轮次 + 投前/投后估值 |
| G | 已投机构 | 按时间列轮次、投资方、金额；缺金额不补猜 |
| H | 本轮投前估值 | 明确投前/参考估值；口径不明时写“公司表示……” |
| I | 本次融资额（或投后估值） | 优先写融资额；仅披露投后时注明“投后” |
| J | 本次融资截止时间 | 明确日期或管理层计划；无信息留空 |
| K | 主营业务 | 具体产品/解决方案级短语 |
| L | 价值 | 七段式项目摘要 |
| M | 收入 | 最近完整年度实际收入；必要时含当年预计 |
| N | 利润 | 与 M 同期的净利润/亏损 |
| O | 项目来源和录入时间 | `项目来源：...\n录入时间：YYYY年M月D日` |
| P | 是否通过 | 未经用户明确判断留空 |
| Q | 否决原因 | 未经用户明确判断留空 |

## 主营业务

- 推荐长度：8-30 个汉字；复杂多产品公司最多约 45 个汉字。
- 使用“应用场景 + 核心产品”或“核心产品 + 关键品类”。
- 好例子：
  - `刻蚀设备用硅电极、硅环及CVD-SiC部件`
  - `压电MEMS固态散热器`
  - `玻璃模压非球面/自由曲面透镜及光学模组`
- 避免：
  - `半导体材料`
  - `元器件`
  - `测试机`
  - 把所有规划产品堆成长句

## 价值栏七段式

严格保持顺序：

1. `团队`：公司成立与员工概况；3-5 位最关键成员，每人一句，保留与项目直接相关的学历、任职和产业经验。
2. `股权结构`：实控人及最终受益比例；主要创始股东、员工平台、主要机构股东。相同管理人的基金主体在能够确认时合并说明。
3. `产品`：主要产品线、具体产品、量产/供货/送样/在研状态。
4. `技术`：核心工艺、性能指标、认证、专利或 Know-how；宣传性结论需降格。
5. `生产、客户`：基地、产线、现有产能、扩产计划；客户按已量产、送样、接触中分层。
6. `市场`：目标细分市场的年份、区域、规模、增速、国产化率；最多保留 1-3 个最相关数字。
7. `收入`：历史实际、当年预计、未来预测分开，注明确认收入/发货/订单口径。

## 简洁度

- 价值栏目标 900-1600 个汉字；资料不足时更短，不为凑字数扩写。
- 每个子项优先 1-4 条；单条通常不超过 80 个汉字。
- 团队不写完整简历；产品不罗列无关型号；股权不机械抄录全部小股东。
- 用数字和状态词替代宣传形容词。
- 同一事实只出现一次。

## 留空规则

- F-J、M-N 无可靠信息时留空，不写 `0`。
- L 内某一段无可靠信息时写 `暂无可靠披露`，不要编造。
- O 缺项目来源时写 `项目来源：待补充`；录入时间使用实际生成日期。
- P-Q 默认留空。
